静态代码压缩

因为Blog中都是静态页面，基本都可以压缩优化，针对html，css，js，图片进行。
这里没必要用gulp去压缩，配置太繁琐，也没法自动化。
直接使用hexo-all-minifier这个模块，
安装：

npm install hexo-all-minifier –save

增加配置：

all_minifier: true

搞定！

文章唯一链接

hexo-abbrlink

文章字数统计和阅读时长

hexo-symbols-count-time
可以替代老的hexo-wordcount。
